Task Tracker

Task Tracker is a simple task manager, where users can organize their daily responsibilities in one place

Technologies:

    React: For creating a highly interactive user interface
    Redux: To build, store, and manage state containers
    React-Redux: To bind React and Redux
    Node.js: As the JavaScript runtime environment
    Sequelize: A promise-based Node.js ORM for Postgres to serve the model relations
    Heroku: For deployment
