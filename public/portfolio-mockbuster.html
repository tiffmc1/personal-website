Mockbuster

Blockbuster lives on via Mockbuster! Choose from our high selection of movies that are sure to make your movie night one you'll never forget!

Technologies:

    Node.js: As the JavaScript runtime environment
    Express: The backend web application framework
    Sequelize ORM for PostgreSQL: To serve as the database
    React: For creating a highly interactive user interface
    JWT: For user and administrator login and sign-up functionality
    Stripe: Used as the checkout feature
