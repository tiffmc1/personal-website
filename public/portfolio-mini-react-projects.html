Mini React Projects

This is a simple website to display mini projects built with React

Personally, I wanted to have a project like this just for learning purposes, and also to be able to continuously practice React and all of its features in one place

Technologies:

    React v18: For creating fast and responsive UI. Includes modern React practices such as functional components, prop rendering, and hooks
    React-Router v6: To build the routes and links
    Axios: For data fetching
    Firebase: For hosting
